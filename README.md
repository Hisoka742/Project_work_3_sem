**Как запустить ноутбук**
1. Локально (на машине):
   - Установите Python 3.8+ и создайте виртуальное окружение.
   - Установите зависимости: `pip install -r requirements.txt` (см. выше).
   - Откройте `курсовая.ipynb` в Jupyter Notebook / JupyterLab и выполните ячейки по порядку.

2. В Google Colab:
   - Загрузите `курсовая.ipynb` в Google Drive или откройте через `File -> Upload notebook` в Colab.
   - Если нужны дополнительные пакеты, добавьте в ячейку: `!pip install ...` и запустите.
